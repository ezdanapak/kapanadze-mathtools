# рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ - kapanadze-mathtools рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ

## ­ЪЊџ рЃарЃљ рЃљрЃарЃўрЃА рЃћрЃА рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ?

`kapanadze-mathtools` рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў Python рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃЋрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљрЃерЃў.

## ­ЪћД рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 1: pip-рЃўрЃЌ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ
```bash
pip install kapanadze-mathtools
```

### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 2: requirements.txt-рЃЊрЃљрЃю рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ
рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃА `requirements.txt` рЃцрЃљрЃўрЃџрЃў рЃљрЃЏ рЃерЃћрЃЏрЃфрЃЋрЃћрЃџрЃЮрЃЉрЃўрЃЌ:
```
kapanadze-mathtools
```

рЃЏрЃљрЃерЃўрЃю рЃњрЃљрЃБрЃерЃЋрЃў:
```bash
pip install -r requirements.txt
```

## ­ЪЊќ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў

### 1. рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃў

```python
from kapanadze_mathtools import statistics

# рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ
grades = [85, 90, 78, 92, 88]

# рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЦрЃБрЃџрЃљ
average = statistics.mean(grades)
print(f"рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЦрЃБрЃџрЃљ: {average}")

# рЃЏрЃћрЃЊрЃўрЃљрЃюрЃљ
median = statistics.median(grades)
print(f"рЃЏрЃћрЃЊрЃўрЃљрЃюрЃљ: {median}")

# рЃЊрЃўрЃАрЃърЃћрЃарЃАрЃўрЃљ рЃЊрЃљ рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃ«рЃарЃљ
var = statistics.variance(grades)
std = statistics.std_dev(grades)
print(f"рЃЊрЃўрЃАрЃърЃћрЃарЃАрЃўрЃљ: {var:.2f}")
print(f"рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃ«рЃарЃљ: {std:.2f}")
```

### 2. рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃћрЃЉрЃў

```python
from kapanadze_mathtools import geometry

# рЃгрЃарЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў
radius = 10
area = geometry.circle_area(radius)
print(f"рЃгрЃарЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: {area:.2f}")

# рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў
length = 15
width = 8
perimeter = geometry.rectangle_perimeter(length, width)
print(f"рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў: {perimeter}")

# рЃАрЃљрЃЏрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў
base = 12
height = 7
triangle_area = geometry.triangle_area(base, height)
print(f"рЃАрЃљрЃЏрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў: {triangle_area}")
```

## ­Ъј» рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮрЃћрЃЉрЃў

### рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮ 1: рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЦрЃБрЃџрЃћрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў
рЃЊрЃљрЃгрЃћрЃарЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф:
1. рЃЏрЃўрЃўрЃдрЃћрЃЉрЃА рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЦрЃБрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃљрЃА
2. рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЦрЃБрЃџрЃљрЃА
3. рЃўрЃърЃЮрЃЋрЃўрЃА рЃЏрЃћрЃЊрЃўрЃљрЃюрЃљрЃА
4. рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџ рЃњрЃљрЃЊрЃљрЃ«рЃарЃљрЃА

### рЃАрЃљрЃЋрЃљрЃарЃ»рЃўрЃерЃЮ 2: рЃЉрЃљрЃдрЃўрЃА рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃљ
рЃЊрЃљрЃгрЃћрЃарЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃА:
1. рЃЏрЃарЃњрЃЋрЃљрЃџрЃў рЃЉрЃљрЃдрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃА (рЃарЃљрЃЊрЃўрЃБрЃАрЃў = 5рЃЏ)
2. рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃљ рЃЉрЃљрЃдрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃА (10рЃЏ x 6рЃЏ)
3. рЃарЃЮрЃЏрЃћрЃџрЃў рЃЉрЃљрЃдрЃўрЃљ рЃБрЃцрЃарЃЮ рЃЊрЃўрЃЊрЃў?

## ­ЪљЏ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃџ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА:

```python
from kapanadze_mathtools import geometry

# рЃћрЃА рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃћрЃЋрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃА
try:
    area = geometry.circle_area(-5)  # рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃарЃљрЃЊрЃўрЃБрЃАрЃў!
except ValueError as e:
    print(f"рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ: {e}")
```

## ­ЪЊІ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

### Statistics рЃЏрЃЮрЃЊрЃБрЃџрЃў:
- `mean(numbers)` - рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃљрЃарЃўрЃЌрЃЏрЃћрЃбрЃўрЃЎрЃБрЃџрЃў
- `median(numbers)` - рЃЏрЃћрЃЊрЃўрЃљрЃюрЃљ
- `variance(numbers)` - рЃЊрЃўрЃАрЃърЃћрЃарЃАрЃўрЃљ
- `std_dev(numbers)` - рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃ«рЃарЃљ

### Geometry рЃЏрЃЮрЃЊрЃБрЃџрЃў:
- `circle_area(radius)` - рЃгрЃарЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў
- `circle_circumference(radius)` - рЃгрЃарЃўрЃА рЃњрЃљрЃарЃерЃћрЃЏрЃЮрЃгрЃћрЃарЃўрЃџрЃЮрЃЉрЃљ
- `rectangle_area(length, width)` - рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў
- `rectangle_perimeter(length, width)` - рЃЏрЃљрЃарЃЌрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃърЃћрЃарЃўрЃЏрЃћрЃбрЃарЃў
- `triangle_area(base, height)` - рЃАрЃљрЃЏрЃЎрЃБрЃЌрЃ«рЃћрЃЊрЃўрЃА рЃцрЃљрЃарЃЌрЃЮрЃЉрЃў

## ­ЪњА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў

1. **рЃўрЃЏрЃърЃЮрЃарЃбрЃў**: рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃерЃў рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћ import
2. **рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ**: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ `help()` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА
3. **рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ**: рЃАрЃфрЃљрЃЊрЃћ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃЌ

```python
# рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃќрЃћ
help(statistics.mean)
```

## РЮЊ рЃ«рЃерЃўрЃарЃў рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃў

**рЃе: рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃЮ, рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЊрЃљ рЃЌрЃБ рЃљрЃарЃљ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ?**  
рЃъ: рЃњрЃљрЃБрЃерЃЋрЃў Python-рЃерЃў:
```python
import kapanadze_mathtools
print(kapanadze_mathtools.__version__)
```

**рЃе: рЃарЃљ рЃЋрЃЦрЃюрЃљ, рЃЌрЃБ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃЏрЃўрЃЋрЃўрЃдрЃћ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃўрЃАрЃљрЃА?**  
рЃъ: рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃў, рЃарЃЮрЃЏ pip рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃЦрЃЋрЃА:
```bash
python -m pip install --upgrade pip
```

## ­Ъџђ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў

рЃарЃЮрЃфрЃљ рЃљрЃЏ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљрЃА рЃўрЃАрЃгрЃљрЃЋрЃџрЃў, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ:
1. рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљ
2. PyPI-рЃќрЃћ рЃњрЃљрЃЏрЃЮрЃљрЃЦрЃЋрЃћрЃДрЃюрЃЮ
3. рЃАрЃ«рЃЋрЃљ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃА рЃњрЃљрЃБрЃќрЃўрЃљрЃарЃЮ
